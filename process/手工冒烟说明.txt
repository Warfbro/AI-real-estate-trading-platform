【手工冒烟说明（微信开发者工具）】

目标：在不依赖旧兼容页面的前提下，验证当前 9 个页面版本下的关键链路、Smoke Helper、继续路由回退和日志留痕。

一、最快命令序列
1. Console 执行：
```js
SMOKE_HELPER.runQuickDemo()
```
2. 可选读取返回值：
```js
const quick = SMOKE_HELPER.runQuickDemo()
console.log(quick.ok, quick.reason_code, quick.duration_ms)
console.log(quick.step_results, quick.navigate_result)
```

返回字段重点：
- `reason_code`
- `step_results`
- `started_at/finished_at/duration_ms`

返回字段速查：
- `ok=true && reason_code="ok"`：一键演示准备成功
- `step_results.*.status`：用于快速判断哪一步失败
- `navigate_result.status`：用于判断是否需要手动跳转
- `seed_base_failed/seed_high_risk_failed/switch_role_failed/verify_high_risk_failed/navigate_failed`

失败态演示模板：
```js
const quickFail = SMOKE_HELPER.runQuickDemo({ role: "guest", navigate: false })
console.log(quickFail.ok, quickFail.reason_code)
console.log(quickFail.step_results.switch_role)
```

预期：
- `quickFail.ok === false`
- `quickFail.reason_code === "switch_role_failed"`

二、基础准备
1. 打开微信开发者工具并确保项目编译成功。
2. 将 `process/devtools_smoke_helper.js` 全量粘贴到 Console 执行。
3. 执行：
```js
SMOKE_HELPER.seedBase()
SMOKE_HELPER.printSummary()
```

预期：
- `buyer_intakes=1`
- `listings=2`
- `comparison_reports=1`
- `risk_checks=1`
- `next_actions=0`
- `advisor_leads=0`

三、AI 路径与高风险上下文
1. 执行：
```js
SMOKE_HELPER.seedHighRisk()
SMOKE_HELPER.verifyHighRiskSeed()
```
2. 进入 AI 页：
```js
wx.navigateTo({ url: "/pages/ai/index?source=risk" })
```
3. 在 AI 页继续完成 manual_review / send_report 相关操作。

预期：
- `SMOKE_HELPER.verifyHighRiskSeed()` 返回 `true`
- AI 页可正常打开
- 当前上下文可触发高风险相关建议

四、经纪人入口与演示跳转
1. 执行：
```js
SMOKE_HELPER.switchRole("admin")
```
2. 手动跳转到当前保留的经纪人入口：
```js
wx.navigateTo({ url: "/pages/import/index?source=broker" })
```

预期：
- 角色切换成功
- 可进入 `/pages/import/index?source=broker`
- 页面标题和副标题按经纪人模式展示

五、日志与状态自检
1. 关键日志名称需可观察到：
- `lead_status_update`
- `manual_review_note_update`
2. 预约线索状态全流转检查顺序：
- `new -> in_progress -> pending_confirmed -> booked -> completed -> closed`
3. manual_review 线索至少检查：
- note 已写入
- updated_at 已刷新
- 状态变更后有 `lead_status_update`

六、推荐命令清单
```js
SMOKE_HELPER.seedBase()
SMOKE_HELPER.seedHighRisk()
SMOKE_HELPER.verifyHighRiskSeed()
SMOKE_HELPER.switchRole("admin")
SMOKE_HELPER.runQuickDemo()
```

七、结束与清理
```js
SMOKE_HELPER.toUser()
SMOKE_HELPER.clearAll()
```
